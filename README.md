# Footy Ball
### Get Near Real-Time Football League Standings

# MVC Architecture Based Minimal Designed App
### Works with the following leagues
- Premiere League (England)
- La Liga (Spain)
- Serie A (Italy)
- Bundesliga (Germany)
- Ligue 1 (France)
- Eredevisie (Netherlands)
